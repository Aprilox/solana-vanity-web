# solana-vanity-web

[![License: CC BY-NC-SA 4.0](https://licensebuttons.net/l/by-nc-sa/4.0/88x31.png)](https://creativecommons.org/licenses/by-nc-sa/4.0/)

> **Solana Vanity Generator** – WebAssembly + Web Workers  
> Préfixe **case-sensitive** (`AaA` → `AaA...`) • **1.2M+ clés/s**  
> UI dark Solana • Console live • Export Phantom-compatible  
> **100% client-side** • **Open-source**

---

### Fonctionnalités
- Préfixe **exact** (majuscules/minuscules respectées)  
- Multi-thread (jusqu’à 256 threads)  
- Console terminal pro (couleurs, threads live)  
- Boutons **Copier** (adresse + clé privée)  
- Export `wallet.json` (Phantom) + backup  
- Popup sécurité à la fermeture  
- Aucune donnée envoyée

---

### Utilisation
1. Tape ton préfixe (ex: `AaA`)  
2. Choisis le nombre de threads  
3. Clique **Lancer**  
4. Copie ou télécharge
